# SU120 PCB Order Guide

## When ordering from Elecrow

https://www.elecrow.com/pcb-manufacturing.html

Use [su120-gerber-elecrow.zip](https://github.com/e3w2q/su120-keyboard/blob/master/su120-gerber-elecrow.zip?raw=true) as the gerber file.

The dimensions are 100 x 100 mm.

The cost does not change between 5 and 10 sheets.
 
Select your favorite color for resist color.

![Elecrow](elecrow.png?raw=true)


## When ordering from FusionPCB

https://www.fusionpcb.jp/prototype-pcb-sale.html

Use [su120-gerber.zip](https://github.com/e3w2q/su120-keyboard/blob/master/su120-gerber.zip?raw=true) as the gerber file.

The dimensions are 100 x 100 mm.

When ordering from this link destination, the number of sheets manufactured is fixed at five.

Select your favorite color for resist color.

![FusionPCB](fusionpcb.png?raw=true)
